### <ngl-file>
#### NglFile

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| `[text]` | Description of the file, ie the filename. | string \| TemplateRef | |
| `[iconName]` | Icon to specify the type of attachment, ie `doctype:pdf`. | string | |
