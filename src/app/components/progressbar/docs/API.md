### <ngl-progress-bar>
#### NglProgressBar

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| `[value]` | The percentage value. | number | |
| `[size]` | Size of the progress bar. | 'x-small' \| 'small' \| 'medium' \| 'large' | 'medium' |
| `[variant]` | Variant appearance of the progress bar. | 'circular' | |
